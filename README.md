Overview

This project is a Fraud Detection Dashboard built using Power BI to analyze transaction data and identify suspicious patterns. The dashboard provides interactive insights into transaction volume, amount, device usage, and regional activity.

Tech Stack

Power BI

Power Query

DAX

Data Source

Dataset imported from Kaggle containing transaction details such as amount, device used, network slice, and region.

Business Problem

Financial organizations need efficient tools to monitor large numbers of transactions and quickly identify potential fraud.

Goal

To visualize transaction data and help detect unusual patterns using an interactive dashboard.

Key Insights

Mobile devices account for a large share of transactions.

Some network slices show higher transaction activity.

Regional analysis helps identify high transaction zones.
